# healerHK Portfolio

A modern horizontal portfolio website for `healerHK`, inspired by product-tour storytelling patterns like Microsoft Edge onboarding pages.

## Design direction

- Full-screen horizontal slide navigation instead of normal vertical scrolling.
- Mouse wheel, keyboard arrows, touch/trackpad horizontal scrolling and on-screen controls.
- Dark neon / glassmorphism visual style.
- F1 coding banner hero asset reused from the GitHub profile.
- Project cards for public repos and private case-study summaries.
- Privacy-safe wording for company/client/private work.

## Sections

1. Start / hero
2. About
3. Featured public projects
4. Technical toolbox
5. Private work + GitHub proof
6. Connect

## Run locally

```bash
npm install
npm run dev
```

Open the URL printed by Vite, usually:

```text
http://localhost:5177
```

## Production build

```bash
npm run build
npm run preview
```

Preview URL:

```text
http://localhost:4177
```
